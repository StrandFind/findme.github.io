Telegram


  • search.buzz.im — поиск в публичных каналах и группах, еще найдет публичные сообщения пользователя
  • Telegago — поиск каналов и групп, включая приватные, а так же поиск в Telegraph статьях
  • lyzem.com — поисковик аналогичный buzzim
  • @usinfobot — при поиске по ID найдет username, а так же может показать старое имя аккаунта
  • archive.org — покажет архивированную версию аккаунта (только с публичным username), канала или группы
  • tgstat.com — поиск по публичным сообщениям в каналах
  • @SangMataInfo_bot — история изменения имени аккаунта
  • @TeleSINT_Bot — найдет группы в которых состоит пользователь
  • @creationdatebot — примерная дата создания аккаунта, бот принимает username, но не работает поиск по ID. Для поиска по ID можно переслать сообщение от пользователя
  • @MySeekerBot — поисковик по иранским каналам
  • @get_kontakt_bot — найдет номер телефона аккаунта, бот принимает username и ID
  • @tgscanrobot — найдет группы в которых состоит аккаунт
  • @Quick_OSINT_bot — найдет группы в которых состоит пользователь и номер телефона
  • TelegramOnlineSpy (github.com/Forichok/TelegramOnlineSpy) — лог онлайн активности аккаунта
  • Exgram — поисковая система на основе Яндекса, поиск по 17 сайтам-агрегаторам, находит Telegraph статьи, контакты, приватные и публичным каналы с группам
  • Commentgram — поиск в комментариях к постам
  • Commentdex — поиск в комментариях к постам
  • @UniversalSearchBot — статус бана пользователя ботом combot, показывает количество блокировок, заблокированные сообщения и дату начала
  • @smart_search_3_bot — находит ФИО
